1  About the workshop

🗓️ September 26-28, 2022
🕘 09:00 - 17:00
🌇 Dar-es-Salaam, Tanzania

This workshop targets public health researchers and aims to accompany them in their journey towards Data Science.

1.1 Is this workshop for me?

This workshop is relevant for researchers who answer yes at least to one of the following questions:

  • Do you who want to develop data science projects in public health?
  • Do you wants to learn more about how open and reproducible science approaches can be used in your daily practice?
  • Are you a Stata user (or any other data analysis language) who would like to expand your data analysis skillset with R?
  • Do you want to bridge analyses between data analysis tools (Stata, R or Python) and to more easily collaborate on data analysis with other researchers?

1.2 Learning objectives

There are two complementary aspects of moving into data science:

  1. a mindset about how scientists think and collaborate about data, and
  2. a skillset which is composed of an ecosystem of tools (mostly open-source) and practices.

Upon completing the workshop, participants will have gained:

  • exposure to data science approach, tools and collaborative practices
  • learned the basics of working with dynamic documents in RStudio, and how to incrementally incorporate R into your existing data analysis workflows
  • hands-on experience on how to interface between Stata and R and combine them depending on your skills and needs. The idea is not to replace everything you are already doing in Stata into R but that you can continue your learning journey after this workshop at your own pace, also using some of the resources listed in Appendix D.

1.3 To do before the workshop

  1. Fill out the online pre-workshop questionnaire

  2. If you have not it yet, please go through the following five steps (about 30 minutes).

    1. Install R (Section A.1)

    2. Install and setup RStudio Desktop (Section A.2)

    3. Install Quarto (Section A.3)

    4. Install the rmarkdown R package (Section A.4)

    5. Create a GitHub account (Section B.1)

    6. Install GitHub Desktop (Section B.2)

Note
  • All software used in this workshop are free.
  • If you have any difficulties with the installation, support can be provided on the first day of the workshop before the first session or during breaks.
Important

For those who have already been introduced to Git, R/RStudio and RMarkdown, note that you need to additionally install Quarto for this workshop.

1.4 Venue

The workshop takes place at Protea Hotel Dar es Salaam Courtyard on Seaview Ocean Road. The conference room is Nelson Hall.
